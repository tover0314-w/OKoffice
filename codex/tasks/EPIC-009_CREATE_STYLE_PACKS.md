# EPIC-009 — PDF Creation and Style Packs

## Goal

Implement deterministic document generation.

## Tools

- Markdown to PDF.
- HTML to PDF if feasible.
- Style pack loader.
- Initial style packs.

## Acceptance criteria

- Generate visually polished PDFs.
- Include validation reports.
- Support at least `plain_report`, `business_report_modern`, `resume_modern`, `invoice_clean`.
