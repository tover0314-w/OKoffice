# EPIC-011 — Docs and Project Polish

## Goal

Make the project look like a serious open-source product.

## Tasks

- Rewrite README with implemented commands.
- Add GIF/screenshot placeholders.
- Add examples.
- Add docs status matrix.
- Add troubleshooting.
- Add comparison/status page.

## Acceptance criteria

- A new developer can run first command in under 5 minutes.
- Every stable tool has one CLI and one MCP/API example.
