# EPIC-004 — Render, Text, Metadata

## Goal

Support basic document reading and page rendering.

## Tools

- `pdf.convert.pdf_to_images`
- `pdf.convert.pdf_to_text`
- `pdf.metadata.read`
- `pdf.metadata.update`
- `pdf.metadata.remove`

## Acceptance criteria

- Render produces image artifacts.
- Text extraction returns per-page text.
- Metadata updates validate output.
