# Screenshot and GIF Placeholders

This directory reserves the public launch visual slots. Keep generated media small, local, and free of private documents.

## Planned assets

- `cli-smoke-output.png` - CLI smoke output after `python scripts/smoke.py`.
- `mcp-tool-discovery.png` - MCP tool discovery showing structured tool metadata.
- `rest-api-docs.png` - REST API docs or health check view for the local server.
- `create-agent-workflow.gif` - Short local create-agent workflow from prompt to validated PDF.

## Capture rules

- Use local fixtures or generated example PDFs only.
- Do not include API keys, local usernames, private file paths, or customer PDFs.
- Prefer clean terminal/browser captures with readable JSON evidence.
